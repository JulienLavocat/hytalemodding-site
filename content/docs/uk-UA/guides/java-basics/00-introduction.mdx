Ласкаво просимо до вашої подорожі у світ програмування на Java! Цей посібник створено, щоб ознайомити вас з основами Java.

## Що таке Java?

Java — це потужна об'єктно-орієнтована мова програмування, створена в 1995 році. Це одна з найпопулярніших мов програмування у світі, яка використовується для всього, від мобільних додатків до корпоративного програмного забезпечення. Найважливіше для нас, що це мова, яку Hytale використовує для своїх серверів та API модифікацій.

## Про що ви дізнаєтесь

Цей посібник структурований таким чином, щоб покроково провести вас через:

1. Синтаксис та основи Java
2. Контроль потоку
3. Об'єктно-орієнтоване програмування

Потім ми зосередимося лише на концепціях, специфічних для Hytale. (буде визначено пізніше)

## Налаштування середовища

Перш ніж зануритися в програмування, вам знадобляться деякі необхідні компоненти, які потрібно встановити:

### Java Development Kit (JDK)

Завантажте та встановіть JDK 25 або вище з [офіційного веб-сайту Oracle](https://www.oracle.com/java/technologies/javase-jdk25-downloads.html) або використовуйте альтернативу з відкритим кодом, таку як [AdoptOpenJDK](https://adoptopenjdk.net/).

### Intergrated Development Environment (IDE)

IDE — це програмне забезпечення, яке надає програмістам комплексні засоби для розробки програмного забезпечення. Популярні варіанти для Java:

* [IntelliJ IDEA](https://www.jetbrains.com/idea/)
* [Eclipse](https://www.eclipse.org/)

## Як користуватися цим посібником

1. Читайте статті по порядку - концепції ґрунтуються одна на одній
2. Набирайте приклади - не копіюйте та не вставляйте, набирання тексту допомагає засвоїти матеріал
3. Експериментуйте - змінюйте приклади, щоб побачити, що відбудеться
4. Практикуйтеся - кожна стаття містить вправи
5. Будьте терплячі - програмування - це навичка, яка вимагає часу для розвитку

Розробка модифікацій - це творче вирішення проблем. Ви будете:

* Додавати нові блоки, предмети та істот до Hytale

* Створювати системи, що взаємодіють із ігровим світом
* Ділитися своїми творіннями зі спільнотою
