# Публікація вашого мода на Modtale

[Modtale](https://modtale.net) — це сучасна платформа для обміну та пошуку модів, плагінів та іншого контенту Hytale. Цей посібник проведе вас через весь процес публікації вашого мода.

## Передумови

Перш ніж почати, переконайтеся, що у вас є:

- Зкомпільований файл `.jar` вашого моду
- Високоякісні скріншоти або зображення вашого моду в дії
- Чіткий логотип або значок (рекомендований розмір: 512x512 пікселів)
- Детальний опис функцій вашого моду
- Будь-які залежності або вимоги, задокументовані

## Покроковий процес публікації

### 1. Налаштування облікового запису

Увійдіть у [Modtale](https://modtale.net) зручним для вас способом. Якщо у вас немає облікового запису, спочатку створіть його.

### 2. Почніть створювати

Натисніть кнопку **«Create»** на панелі навігації або перейдіть безпосередньо на [сторінку завантаження](https://modtale.net/upload).

### 3. Виберіть тип контенту

Виберіть відповідний тип контенту для вашої публікації:

- **Server Plugins** – для серверних плагінів та модифікацій Hytale
- **Data Assets** – для користувацьких пакетів даних, таблиць здобичі та іншого контенту на основі даних
- **Art Assets** – для текстур-паків, моделей та іншого художнього контенту
- **Worlds** – Для користувацьких світів та схем Hytale
- **Modpacks** – Для кураторських колекцій модів та ресурсів

![Вибір типу контенту](/assets/guides/modtale_type.png)

### 4. Основна інформація

Заповніть необхідні дані:

- **Назва**: Виберіть чітку, описову назву для вашого моду
- **Короткий опис**: Напишіть короткий опис (1-2 речення), який пояснює, що робить ваш мод
- **Теги**: Додайте відповідні теги, щоб допомогти користувачам знайти ваш мод

![Назва та опис проєкту](/assets/guides/modtale_title.png)

### 5. Налаштування проєкту

Тепер ви перебуваєте на сторінці керування вашим проєктом. Тут ви можете:

#### Завантажити файли

Перейдіть на вкладку **Files** та завантажте свій файл `.jar`:

- Перетягніть файл або скористайтеся кнопкою завантаження
- Додайте примітки до версії з описом змін та нових функцій
- Тип випуску (Release, Beta, Alpha)

![Інтерфейс завантаження файлів](/assets/guides/modtale_upload.png)

#### Налаштуйте ваш проєкт

- **Logo**: Завантажте високоякісний логотип (рекомендовано формат PNG)
- **Screenshots**: Додайте кілька знімків екрана, що показують ваш мод у дії
- **Description**: Напишіть вичерпний опис, використовуючи форматування Markdown
- **Links**: Додайте посилання на ваш вихідний код, сервер Discord або документацію

## Найкращі практики

### Документація

<Callout type="info">
  Завжди додавайте комплексну документацію до опису вашого мода. Користувачі повинні розуміти, що робить ваш мод, як його використовувати та якісь особливі вимоги.
</Callout>

### Примітка версії:

При завантаженні нових версій:

- Чітко документуйте всі зміни та нові функції
- Включити виправлення помилок та відомі проблеми
- Згадайте будь-які критичні зміни або кроки міграції
- Перелічіть нові залежності або зміни вимог

### Візуальний вигляд

- Використовувати високоякісні знімки екрану, які показують функції моду
- Створіть привабливий логотип, який представляє ваш мод
- Розгляньте можливість додавання зображення банера на сторінку вашого проєкту

### Залучення спільноти

- Оперативно реагуйте на коментарі та відгуки
- Розгляньте можливість створення сервера Discord для підтримки спільноти
- Регулярно оновлюйте свій мод на основі відгуків користувачів

## Після публікації

### Моніторинг вашого проекту

- Перевірте аналітику, щоб побачити статистику завантажень
- Відстежуйте коментарі та відгуки
- Слідкуйте за звітами про помилки та запитами на нові функції

### Маркетинг вашого моду

- Поділіться своїм модом у відповідних спільнотах Hytale
- Створюйте демонстраційні відео або навчальні посібники
- Спілкуйтеся з іншими модераторами та творцями

<Callout type="warning">
  Пам'ятайте, що ваш мод потрібно оновлювати для нових версій Hytale та підтримувати сумісність із системою модингу Hytale, щоб максимізувати охоплення вашої аудиторії.
</Callout>

## Отримання допомоги

Якщо у вас виникнуть будь-які проблеми під час процесу публікації:

- Приєднуйтесь до їхньої спільноти Discord для підтримки
- Зверніться безпосередньо до команди Modtale через їхні канали підтримки
