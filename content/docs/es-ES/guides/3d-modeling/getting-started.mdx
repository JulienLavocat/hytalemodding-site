Empecemos con los fundamentos del modelado 3D.

## Descargar e instalar Blockbench

Hytale utiliza Blockbench para todas las tareas de modelado 3D. Puedes descargarlo desde el sitio web oficial: [Blockbench](https://www.blockbench.net/).

Una vez descargado, sigue las instrucciones de instalación para tu sistema operativo.

<Callout>
  También puedes usar la [versión web](https://web.blockbench.net/) de Blockbench si prefieres no instalar nada, aunque no se recomienda para modelos grandes/complejos.
</Callout>

<Callout type="warning">
  Si estás usando **Linux**, elige el paquete soportado por tu distribución de Linux actual. Si no estás seguro, puedes descargar el AppImage de Blockbench (Recomendado), o el paquete Flatpak, ambos son instaladores universales y funcionan en todas las distribuciones de Linux.
</Callout>

Una vez que termines de descargar e instalar el paquete, verás el siguiente menú en el primer lanzamiento. Lo que indica que has instalado Blockbench exitosamente.

![Blockbench HomePage](https://cdn.internal.hytalemodding.dev/assets/guides/blockbench_homepage.png)

Ahora necesitamos instalar los plugins de Hytale para Blockbench. Para ello, sigue estos pasos:

1. Haz clic en Archivo en la esquina superior izquierda.
2. Selecciona "Plugins"
3. Verás un plugin llamado Hytale Models, si no sale, búscalo y selecciónalo.
   ![Blockbench Plugins Overview](https://cdn.internal.hytalemodding.dev/assets/guides/blockbench_plugins_overview.png)
4. Haz clic en **Instalar**.

¡Ya está! Ahora vamos a crear un modelo. Si estás creando un modelo de personaje, como un PNJ, animales, etc. deberías usar el modelo de tipo **Hytale Character**. Para esta guía, nos centraremos en la creación de un _prop_ de Hytale, pero es bastante similar. Haz clic en el modelo de tipo **Hytale Prop**.

![Blockbench Hytale Character](https://cdn.internal.hytalemodding.dev/assets/guides/blockbench_hytale_character.png)

Ahora se te pedirá introducir un nombre de archivo, escribe el nombre de tu modelo.

<Callout type="">
  No añadas caracteres especiales, espacios, símbolos o caracteres no latinos (excepto guiones bajos).

  ✅ mi_modelo

  ❌ مi m♥פel$
</Callout>

![Blockbench Hytale Character](https://cdn.internal.hytalemodding.dev/assets/guides/blockbench_hytale_character_new_project.png)

¡Eso es todo! ¡Ahora tienes un nuevo modelo de personaje de Blockbench para Hytale! En nuestras próximas guías aprenderemos a crear como tal este modelo.
