Agora que você criou o seu mod, chegou a hora de compartilhá-lo para todos! Existem diversas plataformas que permitem que você publique seu mod, como Modtale, CurseForge, etc.

Nós compilamos guias de publicação para cada plataforma, então você pode facilmente disponibilizar seu mod  nas mãos dos jogadores, não importando o mercado que eles usem.

- [Publicando em Modtale](./modtale.mdx)

<Callout>
  Se você é proprietário(a) de uma plataforma de publicação de mods, contate-nos pelo e-mail `hello@hytalemodding.dev` para listá-lo em nossa documentação.
</Callout>
