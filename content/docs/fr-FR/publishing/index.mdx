Maintenant que vous avez créé votre mod, il est temps de le partager avec le monde entier ! Il y a beaucoup de places de marché qui vous permettent de publier votre mod, tels que Modtale, CurseForge, etc.

Nous avons compilé des guides à publier pour chaque marché, pour que vous puissiez facilement mettre votre mod entre les mains des joueurs, peu importe la place de marché qu'ils utilisent.

- [Publier sur Modtale](./modtale.mdx)
