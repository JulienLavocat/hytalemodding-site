# Publier sur Modtale

Modtale est une plateforme moderne dédiée au partage et à la découverte de mods, plugins et autres contenus pour Hytale. Ce guide vous guidera dans la publication complète de votre mod.

## Condition Préalable

Avant de commencer, assurez-vous d'avoir :

- Un fichier compilé `.jar` de votre mod
- Des captures d'écran ou des images de haute qualité de votre mod en action
- Un logo ou une icône claire (taille recommandée : 512x512px)
- Une description détaillée des fonctionnalités de votre mod
- Toutes les dépendances ou prérequis documentés

## Processus de publication étape par étape

### 1. Configuration de compte

Connectez-vous à [Modtale](https://modtale.net) en utilisant la méthode que vous préférez. Si vous n'avez pas de compte, créez en un d'abord.

### 2. Créer maintenant

Cliquez sur le bouton **"Créer"** dans la barre de navigation ou allez directement sur la [page de téléchargement](https://modtale.net/upload).

### 3. Choisir le type de contenu

Sélectionner le type de contenu approprié pour votre soumission :

- **Plugins Serveur** - Pour les plugins et modifications côté serveur Hytale
- **Data Assets** - Pour les packs de données personnalisés, les tables de butin et d'autres contenus axés sur les données
- **Actifs artistiques** - Pour les packs de textures, modèles et autres contenus artistiques
- **Mondes** - Pour les mondes et schémas personnalisés d'Hytale
- **Modpacks** - Pour les collections organisées de mods et de ressources

![Sélection de type de contenu](/assets/guides/modtale_type.png)

### 4. Informations générales

Remplissez les informations essentielles :

- **Nom** : Choisissez un nom clair et descriptif pour votre mod
- **Résumé** : Écrivez une brève description (1-2 phrases) qui explique ce que fait votre mod
- **Tags** : Ajouter des tags pertinents pour aider les utilisateurs à trouver votre mod

![Titre et description du projet](/assets/guides/modtale_title.png)

### 5. Configuration du Projet

Maintenant vous êtes sur la page de gestion de votre projet. Ici vous pourrez :

#### Ajouter des fichiers

Naviguez dans l'onglet **"Fichiers"** et téléchargez votre fichier `.jar`:

- Glissez-déposez votre fichier ou utilisez le bouton de téléchargement
- Ajouter des notes de version décrivant les modifications et les nouvelles fonctionnalités
- Marquer le type de version (Release Beta, Alpha)

![Interface de chargement de fichiers](/assets/guides/modtale_upload.png)

#### Personnaliser votre projet

- **Logo** : Téléchargez un logo de haute qualité (format PNG recommandé)
- **Captures d'écran**: Ajoute plusieurs captures d'écran montrant votre mod en action
- **Description** : Écrire une description complète en utilisant le formatage Markdown
- **Liens** : ajoutez des liens vers votre code source, serveur Discord, ou documentation

## Bonnes pratiques

### Documentation

<Callout type="info">
  Toujours inclure une documentation complète dans la description de votre mod. Les utilisateurs doivent comprendre ce que fait votre mod, comment l'utiliser, et toutes les exigences spéciales.
</Callout>

### Notes de version

Lors du téléchargement de nouvelles versions :

- Documentez clairement toutes les modifications et les nouvelles fonctionnalités
- Inclure les corrections de bugs et les problèmes connus
- Mentionnez les changements ou les étapes de migration
- Lister les nouvelles dépendances ou modifications de conditions

### Appel visuel

- Utilisez des captures d'écran de haute qualité qui mettent en valeur les fonctionnalités de votre mod
- Créez un logo qui attire l'attention et qui représente votre mod
- Envisagez d'ajouter une image de bannière pour votre page de projet

### Engagement envers la communauté

- Répondez rapidement aux commentaires et aux retours
- Envisagez de créer un serveur Discord pour le support de la communauté
- Mettre à jour régulièrement votre mod en fonction des commentaires des utilisateurs

## Après publication

### Suivi de votre projet

- Consultez les statistiques de téléchargement
- Surveiller les commentaires et les avis
- Garder une trace des rapports de bugs et des demandes de fonctionnalités

### Marketing de votre mod

- Partagez votre mod sur des communautés Hytale pertinentes
- Créer des vidéos de vitrine ou des tutoriels
- Engagez avec d'autres modders et créateurs

<Callout type="warning">
  N'oubliez pas de garder votre mod à jour pour les nouvelles versions de Hytale et de maintenir la compatibilité avec le framework de modding d'Hytale pour maximiser la portée de votre public.
</Callout>

## Obtenir de l'aide

Si vous rencontrez des problèmes durant le processus de publication:

- Rejoignez la communauté Discord pour obtenir de l'aide
- Contactez l'équipe Modtale directement via leurs canaux de support
