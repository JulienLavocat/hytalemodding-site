Commençons par les bases de la modélisation 3D.

## Télécharger et installer Blockbench

Hytale utilise des Blockbench pour toutes les tâches de modélisation 3D. Vous pouvez le télécharger à partir du site officiel [Blockbench](https://www.blockbench.net/).

Une fois téléchargé, suivez les instructions d'installation pour votre système d'exploitation.

<Callout>
  Vous pouvez également utiliser la [version web](https://web.blockbench.net/) de Blockbench si vous préférez ne rien installer, bien que ce ne soit pas recommandé pour les modèles volumineux ou compliqués.
</Callout>

<Callout type="warning">
  Si vous utilisez **Linux**, sélectionnez le paquet supporté par votre distribution Linux actuelle. Si vous n'êtes pas sûr, vous pouvez télécharger Blockbench AppImage (Recommandé), ou le paquet Flatpak, tous deux sont des installateurs universels et fonctionnent sur toutes les distributions Linux.
</Callout>

Une fois que vous avez fini de télécharger et d'installer le paquet, vous verrez le menu ci-dessous au premier lancement. Ce qui indique que vous avez installé Blockbench avec succès.

![Page d'accueil de Blockbench](https://cdn.internal.hytalemodding.dev/assets/guides/blockbench_homepage.png)

Maintenant, nous devons installer les plugins d'Hytale Blockbench, pour cela, suivez ces étapes :

1. Cliquez sur "Fichier" en haut à gauche.
2. Sélectionnez "Plugins".
3. Dans ce menu, vous allez voir le plugin "Hytale Models". Si vous ne le voyez pas, recherchez-le et sélectionnez-le.
   ![Vue d'ensemble des plugins Blockbench](https://cdn.internal.hytalemodding.dev/assets/guides/blockbench_plugins_overview.png)https://cdn.internal.hytalemodding.dev/assets/guides/blockbench_plugins_overview.png
4. Cliquez sur **Installer**

Et voilà, vous êtes prêt ! Passons à la suite avec la création d'un modèle. Si vous voulez créer un modèle de personnage, comme un NPC (personnage non-joueur), un animal, etc. vous devez utiliser le type de modèle **Hytale Character**. Click on the **Hytale Prop** model Type. For this guide, we will focus on creating the Hytale Prop, but it is pretty similar.

![Blockbench Hytale Character](https://cdn.internal.hytalemodding.dev/assets/guides/blockbench_hytale_character.png)

Maintenant on vous demandera de saisir un nom de fichier, entrez le nom de vos modèles.

<Callout type="warning">
  N'ajoutez pas de caractères spéciaux, espaces, symboles ou caractères non latins (à l'exception des tirets bas).

  ✅ mon_modele

  ❌ مOw m♥פel3
</Callout>

![Blockbench Hytale Character](https://cdn.internal.hytalemodding.dev/assets/guides/blockbench_hytale_character_new_project.png)

C'est ça ! Vous avez maintenant un nouveau modèle de personnage grâce à "Hytale Blockbench" ! Dans nos prochains guides, nous apprendrons à créer ce modèle.
