# Вносить вклад в HytaleModding

Мы приветствуем вклад сообщества для улучшения и расширения документации HytaleModding Ваш любой вклад значим: Исправление опечаток, добавление руководств, примеры кодов

## Как внести вклад

### Сообщить о проблеме

Если вы нашли проблему с документацией, то, пожалуйста, сообщите об этом, открыв новую проблему в нашем [GitHub репозитории](https://github.com/HytaleModding/site/issues).

Если вы можете работать с проблемой, то вас могут попросить быть назначенным на это и подтвердить PR

### Отправка Pull Request'ов

1. Форкнуть репозиторий на GitHub.
2. Клонируйте свой форк на вашу локальную машину.
3. Создать новую ветку для ваших изменений

```bash
git checkout -b my-feature-branch
```

4. Сделайте ваши изменения и отправьте им commit с чистым сообщением. Мы следуем [Обычным commit](https://www.conventionalcommits.org/en/v1.0.0/), ваш PR не будет принят, если ваши сообщения для commit не будут следовать этим условиям.

```bash
git commit -m "Add feature X to documentation"
```

5. Внесите ваши изменения в ваш форк

```bash
git push origin my-feature-branch
```

6. Откройте пул запроса на оригинальном репозитории, описывая ваши изменения и почему они должны быть перенесены
